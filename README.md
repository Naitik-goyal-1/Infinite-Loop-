# ♾️ Infinite Loop Animation

A simple and creative Infinite Loop Animation built using pure HTML.

## 🚀 Overview

This project demonstrates how a continuous looping animation can be created using only HTML and basic web concepts. The goal was to experiment with visual effects and create an endlessly repeating animation without relying on external frameworks or libraries.

## ✨ Features

- Infinite looping animation
- Lightweight and fast
- Pure HTML implementation
- Beginner-friendly code structure
- Easy to customize and extend

## 🛠️ Technologies Used

- HTML5
